## User Stories

```gherkin
Feature: 음성 통화
  As a 팬 사용자
  I want to 좋아하는 연예인에게 전화를 걸고 싶다
  So that 실제로 대화하는 듯한 경험을 느낄 수 있다

  Scenario: 첫 통화
    Given 사용자가 Phone 앱을 열었다
    And GD의 연락처를 선택했다
    When 통화 버튼을 누른다
    Then 전화 연결 애니메이션이 표시된다
    And 2초 내에 GD의 목소리로 "여보세요?"가 들린다
    And 사용자가 말하면 STT로 변환된다
    And GD 페르소나로 응답이 생성된다
    And TTS로 음성이 재생된다

Feature: 문자 메시지
  As a 팬 사용자
  I want to 연예인과 문자를 주고받고 싶다
  So that 일상적인 대화를 나눌 수 있다

  Scenario: 문자 대화
    Given 사용자가 Messages 앱을 열었다
    And GD와의 대화방을 선택했다
    When "오늘 뭐 해?"라고 입력하고 전송한다
    Then 타이핑 인디케이터가 표시된다
    And 2초 내에 GD 스타일의 답장이 도착한다
    And 답장에 읽음 표시가 생긴다

Feature: 선제적 연락
  As a 팬 사용자
  I want to 연예인이 먼저 연락하길 원한다
  So that 특별한 관계처럼 느낄 수 있다

  Scenario: 아침 인사
    Given 사용자가 앱을 설치하고 3일이 지났다
    And 아침 9시가 되었다
    When 시스템이 트리거를 발동한다
    Then 푸시 알림 "GD: 잘 잤어? ㅎㅎ"가 전송된다
    And 사용자가 앱을 열면 Messages에서 대화를 이어갈 수 있다

Feature: 🆕 실시간 사진 공유 (Live Photo)
  As a 팬 사용자
  I want to "지금 뭐해?" 질문에 연예인이 사진을 보내주길 원한다
  So that 실제로 연예인의 일상을 엿보는 듯한 친밀감을 느낄 수 있다

  Scenario: 오후에 "지금 뭐해?" 질문
    Given 사용자가 GD와 채팅 중이다
    And 현재 시간이 오후 3시이다
    And GD의 스케줄에 "녹음"이 등록되어 있다
    When 사용자가 "지금 뭐해?"라고 입력한다
    Then 시스템이 의도를 감지한다
    And 녹음실 컨텍스트로 이미지가 생성된다
    And 3초 내에 [녹음실 사진] + "녹음 중이야~" 메시지가 도착한다
    And 사진은 1인칭 시점 (GD가 찍은 것처럼)이다
    And 얼굴은 포함되지 않는다

  Scenario: 밤에 "뭐하고 있어?" 질문
    Given 사용자가 GD와 채팅 중이다
    And 현재 시간이 밤 11시이다
    And GD의 스케줄에 특별한 일정이 없다
    When 사용자가 "뭐하고 있어?"라고 입력한다
    Then 집에서 휴식하는 컨텍스트로 이미지가 생성된다
    And [야경이 보이는 창가 사진] + "집에서 쉬고 있어 ㅎㅎ" 메시지가 도착한다
```

## Acceptance Criteria Checklist

- [ ] iPhone UI가 실제 폰처럼 보이고 느껴지는가?
- [ ] 60fps 애니메이션이 유지되는가?
- [ ] 음성 통화 시 500ms 내 첫 응답이 오는가?
- [ ] 문자 응답이 2초 내에 도착하는가?
- [ ] 페르소나가 일관되게 유지되는가?
- [ ] 이전 대화 내용을 기억하는가?
- [ ] SNS 피드가 실제 데이터를 보여주는가?
- [ ] 푸시 알림이 정상 전송되는가?
- [ ] 부재중 전화 → 콜백 시나리오가 작동하는가?
- [ ] 🆕 "지금 뭐해?" 질문에 상황 맞는 이미지가 생성되는가?
- [ ] 🆕 이미지 생성이 3초 내에 완료되는가?
- [ ] 🆕 생성된 이미지에 얼굴이 포함되지 않는가?
